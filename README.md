# CPToolKit

A personal Competitive Programming Toolkit built in C++ to improve problem-solving speed, reduce repetitive coding, and maintain reusable implementations of common algorithms and data structures.

---

## Overview

CPToolKit is a structured collection of templates, algorithms, and utilities designed for efficient coding during competitive programming contests such as Codeforces, LeetCode, and ICPC.

The objective is to minimize boilerplate code and focus more on problem-solving logic.

---

## Features

- Fast I/O template
- Ready-to-use code snippets
- Optimized data structures
- Modular and reusable functions
- Clean and readable C++ implementations

---

## Contents

### Templates
- Fast I/O
- Debug macros
- Common typedefs

### Algorithms
- Sorting and Searching
- Recursion and Backtracking

### Data Structures
- Arrays and Vectors
- Stacks and Queues
- Trees and Graphs

- Mathematical functions
